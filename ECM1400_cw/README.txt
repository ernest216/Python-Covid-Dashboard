README
## General info
 Implementing a simple Covid Dashoard using Python and Flask, the dashboard application will coordinate information about the COVID infection rates from the Public Health England API and news
 stories about Covid from a given news API.
## Prerequesites
 Project is created with:
 * Python version: 3.10
## installation
 Use the package manager [pip] to install flask.
 * $ pip install Flask

 The Covid API in the program supplies the latest data for the COVID-19 outbreak in the United Kingdom. The endpoint for the data provided using this SDK is:

 https://api.coronavirus.data.gov.uk/v1/data
 To install, please run:

 * pip install uk-covid19
 You may install the library for a specific version of Python as follows:

 * python -m pip install uk-covid19
 A Python client for the News API is used. This is a Python client library for News API V2. The functions and methods for this library should mirror the endpoints specified by the News API documentation.
 Installation for the package can be done via pip:

 * $ python -m pip install newsapi-python
## Getting_started_tutorial
 Run the code and then click on the link formed by the python environment to access the dashboard, by command+click, then type in index at the end of the link and you can see the up-to-date dashboard
